팀 공유 동기화 (GitHub Gist)

        
        
      

      
연결하면 팀 전체가 같은 말투 스타일·참고파일·프로젝트를 공유합니다. GitHub Gist를 공용 저장소로 사용합니다 — Firebase 불필요.

      

        설정 방법 (2단계)

        ① gist.github.com → 새 Gist 생성 → 파일명 translatehub.json, 내용 {} → Create public gist

        ② GitHub 토큰 발급 → Gist 권한만 체크 → 토큰 복사

        → 아래에 입력 후 팀원들과 공유하면 끝!
      

      

        
      

      
gist.github.com 주소의 마지막 영문+숫자 조합 (예: a1b2c3d4e5f6...)

      

        
      

      
팀원 모두 같은 토큰 사용 가능 (내부 도구이므로 공유 OK)

      
      
ℹ️ API 키(Claude/OpenAI)는 개인 브라우저에만 저장되고 Gist에는 올라가지 않습니다.
