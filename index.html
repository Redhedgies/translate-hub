      연동 방법 안내

      • Discord: 서버 채널 설정 → 연동 → 웹훅 만들기에서 URL 복사

      • Facebook: Meta for Developers에서 페이지 액세스 토큰 발급

      • Twitter/X: Zapier에서 Webhook → Post Tweet Zap 생성 후 URL 사용

      • Instagram: Zapier에서 Webhook → Create Media Post Zap 생성 후 URL 사용
